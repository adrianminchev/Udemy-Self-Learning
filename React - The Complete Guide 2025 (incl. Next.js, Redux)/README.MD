# React – The Complete Guide (incl. Next.js, Redux)

## INTRO
This repository documents my completion of the **“React – The Complete Guide (incl. Redux)”** course.  
The course provides a comprehensive, hands-on introduction to building modern, scalable web applications with **React.js** and related technologies.

---

## OVERVIEW
- Core React concepts (components, JSX, props, state)
- Building interactive UIs with events and hooks
- Managing side effects and application lifecycle
- Styling with CSS Modules and Styled Components
- Handling forms, validation, and HTTP requests
- State management with **Redux & Redux Toolkit**
- Client-side routing with **React Router**
- Authentication and protected routes
- Testing, optimization, and deployment
- Introduction to **Next.js** and TypeScript with React
