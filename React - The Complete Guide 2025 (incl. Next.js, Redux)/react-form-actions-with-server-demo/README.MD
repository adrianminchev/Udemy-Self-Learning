# React Form Action

A demo application with a small Node.js backend server for exploring and testing different form-handling scenarios using Reactâ€™s useActionState form-action hook.

## Getting Started

### 1. Install all project dependencies
# In the main app location
npm install

# In the backend application (for the local BE server)
cd backend
npm install

# Start the frontend (main app)
npm run dev

### 2. Start the application
# Start the backend server
cd backend
node app.js

### 3. Access the project
Once both servers are running, open your browser at:
http://localhost:5173/
