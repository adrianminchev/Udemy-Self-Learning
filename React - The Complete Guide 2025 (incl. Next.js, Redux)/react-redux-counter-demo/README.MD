# React and Redux/Redux toolkit App Demo

A demo application featuring a simple counter and basic authentication components, designed to explore Redux’s main concepts, as well as Redux Toolkit’s createSlice function, which simplifies the overall development process.

## Getting Started

### 1. Install all project dependencies
# In the main app location
npm install

### 2. Start the application
# Start the front-end server
npm start

### 3. Access the project
Once the servers is running, open your browser at:
http://localhost:3000/
