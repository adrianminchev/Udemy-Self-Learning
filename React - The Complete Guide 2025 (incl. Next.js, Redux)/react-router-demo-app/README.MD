# React Roter Demo App
A small, simple demo application for reviewing the key features of React Router, using react-router-dom in a minimal multi-page SPA.

## Getting Started

### 1. Install all project dependencies
# In the main app location
npm install

### 2. Start the application
# Start the front-end server
npm start

### 3. Access the project
Once the servers is running, open your browser at:
http://localhost:3000/
